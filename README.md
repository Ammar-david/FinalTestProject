# FinalTestProject
## Project Description

This project is a QA testing project that includes tests for a website, a mobile application, a SQL database, an electron application, and a computer calculator application.

## Installation

To get started, follow these steps:

1. Install Java preferably version "1.8.0_211".
2. Clone the GitHub repository using the command `git clone https://github.com/Ammar-david/FinalTestProject.git`.
3. Navigate to the project directory using the command `cd FinalTestProject/`.
4. Build the project using Maven with the command `mvn clean install`.

After completing these steps, you will have installed the project and can begin using it.

This will build the project and run all the included tests. If any tests fail, you will see a report of the failures in the console output.

## Usage

To use this project, you can either run the tests individually or as a group. 
To run a single test, use the following command:

1.mvn test -Dtest=TestClassName#testMethodName.

2.Replace "TestClassName" with the name of the test class you want to run, and "testMethodName" with the name of the test method you want to run.

To run all tests in the project, use the following command:

1.mvn test
